# AI Chat History

## AI Tool Used

ChatGPT (OpenAI)

## Purpose of Use

ChatGPT was used as a learning and development support tool during the implementation of this Enterprise Applications project.

The AI was used to:

* Explain Spring Boot concepts
* Explain Spring MVC controllers
* Explain JPA and Hibernate annotations
* Review entity design
* Review repository design
* Assist with Thymeleaf templates
* Help identify and fix errors
* Review project structure
* Assist with validation and form handling
* Assist with README preparation

All suggestions were reviewed, tested and adapted before being integrated into the final application.
